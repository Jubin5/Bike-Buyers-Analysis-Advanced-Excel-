#  Bike Buyers Analysis Dashboard (Advanced Excel)

##  Project Overview
This project analyzes customer demographic and lifestyle data to understand the key factors influencing **bike purchasing behavior**. Using **Advanced Excel techniques**, an interactive dashboard was developed to help identify high-probability customer segments and regional trends.

---

##  Objective
- Analyze customer attributes affecting bike purchase decisions  
- Identify patterns across income, age, region, and commute distance  
- Build an interactive Excel dashboard for business decision support  

---

##  Dataset Description
The dataset contains customer-level information including:

- **Demographics:** Age, Gender, Marital Status, Education  
- **Socio-economic Factors:** Income, Occupation, Home Ownership  
- **Lifestyle Attributes:** Commute Distance, Number of Cars, Region  
- **Target Variable:** Purchased Bike (Yes / No)  

---

##  Tools & Excel Features Used
- Microsoft Excel (Advanced)
- Pivot Tables & Pivot Charts
- Slicers for dynamic filtering
- Logical and aggregation formulas (IF, COUNTIFS, AVERAGE, etc.)
- Data cleaning and transformation
- Interactive dashboard design

---

##  Analysis Performed
- Purchase behavior analysis by **region, age group, income level, and commute distance**
- Comparison of bike buyers vs non-buyers
- Regional trend analysis using pivot charts
- Interactive filtering using slicers for real-time insights

---

##  Key Insights
- Customers with **shorter commute distances** showed higher bike purchase rates  
- **Middle-income and professional segments** were more likely to purchase bikes  
- Bike purchase behavior varied significantly across regions  
- Age group and car ownership influenced customer buying decisions  

---

##  Dashboard Highlights
- Interactive slicers for Region, Gender, and Marital Status  
- Pivot-based charts for fast and dynamic data exploration  
- Clean, business-oriented dashboard layout  

---


---

##  Conclusion
This project demonstrates practical **data analyst skills using Advanced Excel**, including data cleaning, exploratory analysis, dashboard creation, and business insight generation. The dashboard enables stakeholders to quickly identify customer segments and make data-driven marketing decisions.
